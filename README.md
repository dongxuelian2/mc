# Three.js 方块世界原型

一个轻量的 Minecraft 风格原型，基于 Three.js 与 Vite。包含基础的地形生成、方块放置/破坏、粒子与音效、背包和物品栏。

## 功能
- 程序化生成草地/沙地/石头/树木地形
- 邻接剔除的体素网格（减少面数）
- 方块放置与破坏（含粒子碎屑与合成破坏音效）
- 基础第一人称移动、跳跃、重力与碰撞
- 7 种初始方块与简单生成纹理（可替换为自定义材质）
- 热键切换物品栏、背包界面展示/选取方块

## 运行
```bash
npm install
npm run dev
```
浏览器打开提示的本地地址即可。

## 操作
- 鼠标左键：破坏方块
- 鼠标右键：放置当前选中方块
- 滚轮 / 1-7：切换物品栏方块
- WASD：移动，空格：跳跃
- E：打开/关闭背包（会解除指针锁定，点击画面重新锁定）

## 自定义材质
`src/main.js` 中的 `tilePainters` 定义了各方块使用的生成纹理，可修改为载入自定义贴图或扩展新的方块类型。
