这里是可选的方块贴图目录（默认会回退到程序生成的纹理）。

如果你有自己的贴图包（或从本地 Minecraft 资源中提取），把下列文件放到这里即可自动生效：
- `grass_block_top.png`
- `grass_block_side.png`
- `dirt.png`
- `stone.png`
- `sand.png`
- `oak_log.png`
- `oak_log_top.png`
- `oak_leaves.png`
- `oak_planks.png`

提示：不要把 Mojang/Minecraft 的原版资源直接提交到仓库；建议只在本地使用，并保持在 `.gitignore` 中。
